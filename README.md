# E-Kart PHP Project

A **Mini E-Commerce (E-Kart) web application** developed using **Core PHP** and **MySQL**.  
This project demonstrates basic e-commerce functionalities such as product listing, cart management, and order handling.

---

## Project Overview

The **E-Kart PHP Project** is a simple online shopping system built without any framework.  
It is designed to help beginners understand **CRUD operations**, **database connectivity**, and **session handling** in PHP.

---

## Features

- User registration & login
- Product listing
- Add to cart
- Update / remove cart items
- Order placement
- Admin product management (CRUD)
- MySQL database integration
- Session-based cart system

---

## Technologies Used

- Core PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap
